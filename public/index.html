3D Manifest Loader example links
The following links open the 3D Manifest Loader with the Manifest Url input field set to a manifest for a scene containing an asset from the IIIF 3D TSG assets folder.
  1. Astronaut Displays single 3D model with annotation based on model_origin.json prototype.
  2. Saxophone Displays single 3D model with annotation based on model_origin.json prototype.
  3. Nun's cell Displays single 3D model with annotation based on model_origin.json prototype.
  4. Venus de Milo Displays single 3D model with annotation based on model_origin.json prototype.
  5. Whale Displays 2 separate 3D models each placed with model_position.json annotation. Related to Core User Story #12
  6. Archimedes Displays a 3D model along with a supplementing annotation described by an X3D file.
